# Tkinter の機能概要

## ウィジェット変数

### ウィジェット変数とは

 - ウィジェットの状態を制御するために使用されるオブジェクト
 - ウィジェットが表示される際に、ウィジェット変数に格納された値がウィジェットに反映される
 - ユーザーがウィジェットを操作した場合、その変更がウィジェット変数にも反映される

#### よく使われるウィジェット変数

 - StringVar：文字列を保持
 - IntVar：整数を保持
 - DoubleVar：浮動小数点を保持
 - BooleanVar：真偽値を保持

## samples - オプションメニューを使った処理の制御

### optionmenu_basic

- 選択しているメニューの値を取得するプログラム
- 値の取得にフォーカス

### optionmenu_callback

- メニューを選択した時に値を取得するプログラム
- コールバックの使い方を示す

### combobox_linked

- メニューの選択内容で別メニューの選択肢を変えるプログラム
- メニュー連動の基本

## samples - ラジオボタンを使った処理の制御

### radiobutton_basic

-選択したラジオボタンに設定した値を取得する

### radiobutton_conditional

- ラジオボタンに設定した数値で処理を分岐させる
- 選択値による条件分岐

### radiobutton_dict_dispatch

- ラジオボタンに設定した文字列で処理を分岐させる
- 辞書でハンドラを引くディスパッチパターン
